# Inverted-Index

Example commands:
```
-path [path]
-query [path]
-index [path]
-results [path]
```
```
java Driver -path input/index/simple
            -query input/query/simple.txt
            -index index-simple.json
            -results results-simple.json
```
```
java Driver -path input/index/simple
            -query input/query/simple.txt
            -exact -results 
```
